# 😊 Smiley Face – Mini Frontend Project

This is a basic frontend project created using **HTML and CSS**.  
This is a fun little beginner project. It displays a bright and cheerful smiley face built entirely with `div` elements and basic styling — no images, no JavaScript!

---

## 📁 Project Files

- `index.html` – Main HTML file with embedded CSS
- `smiley-image.png` – Screenshot of the smiley face output
- `README.md` – This file

---

## 📸 Output Preview

![Smiley Screenshot](./smiley-image.png)

---

## 🚀 Features

- Simple and clean design using only HTML & CSS
- Rounded face, eyes, and smile created with `border-radius`
- CSS positioning with `margin`, `top`, `left`, and `absolute`

---

## 🧠 What I Practiced

- Using `border-radius` to make shapes
- Creating a layout without external stylesheets
- Positioning elements with `relative` and `absolute`

---

## 🌐 How to Run

> Open the `index.html` file in any web browser — no setup or installation required.
